# ms-office-key
Find office key
- Usage: Run Powershell as administrator
- Set-ExecutionPolicy -ExecutionPolicy RemoteSigned
- Import-Module C:\Users\ $USER\Documents\FindDecodeMSOfficekey.ps1; Get-MSOfficeProductKey
